KomiX
=====

This is a comic viewer.

License
-------

GPLv3 or later

Building Dependency
-------------------

* `CMake`_ >= 2.6
* `Qt toolkit`_ >= 4.5
* `Boost`_

  * `Boost.Signals2`_

Supported Toolchains
--------------------

* `GNU Compiler Collection`_ >= 4.6
* `LLVM Clang`_ >= 3.1
* `Microsoft Visual C++`_ >= 10
